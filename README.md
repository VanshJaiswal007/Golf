# Golf Charity Subscription Platform

Built to match the PRD scope:
- subscription engine
- score management with rolling last-5 logic
- monthly draw and reward system
- charity selection and contribution logic
- user dashboard
- admin dashboard
- winner verification and payout tracking

## Default admin login

Email: `admin@golfcharity.local`  
Password: `Charity@12345`

## Main routes

User Dashboard: `/dashboard`  
Admin Panel: `/admin`

## Run locally

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the app:
   ```bash
   npm run dev
   ```

The app seeds its own data store on first run.
